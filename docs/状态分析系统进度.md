## 关键设计决策

| 决策项 | 方案 | 状态 |
|--------|------|------|
| 响应模式 | immediate / mention / buffer | 已确定 |
| ContextAnalyzer 模型 | gpt-4o-mini（轻量级） | 已确定 |
| StateAnalyzer 模型 | gpt-4o（深度分析） | 已确定 |
| 状态分析频率 | 每5条消息分析一次 | 已确定 |
| 缓冲区最大大小 | 50条消息 | 已确定 |
| 缓冲区消息最大年龄 | 300秒（5分钟） | 已确定 |

基于 KiraAI 的插件架构设计：

```
core/plugin/
├── __init__.py
├── plugin.py              # BasePlugin 抽象基类
├── plugin_context.py      # 插件上下文
├── plugin_registry.py     # 插件注册与管理
├── plugin_handlers.py     # 事件处理器
└── builtin_plugins/       # 内置插件示例
    ├── default/           # 默认插件
    ├── search/            # 搜索插件
    ├── memory/            # 记忆插件
    ├── file/              # 文件处理插件
```
